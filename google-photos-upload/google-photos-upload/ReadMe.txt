---------------------------------------
|                                     |
|	Google Photos Upload              |
|	by Rune Rasmussen                 |
|                                     |
---------------------------------------

This is an open source desktop tool to upload a local image directory into an Album in Google Photos, on pc/mac/linux.

Find more information, a user guide and the source code online:
https://github.com/runerasmussen/google-photos-upload/
